# ASSA

## Outline

❤️‍🔥트렌드가 궁금할 때, **ASSA**와 함께하세요!

- **ASSA는 당신이 트렌드에 뒤처지지 않도록 돕는 큐레이션 앱입니다.**
- 매시간 업데이트되는 최신 이슈와 핫한 트렌드를 빠르고 간편하게 확인하세요. 
복잡한 정보는 줄이고, 놓치기 쉬운 트렌드를 딱 맞는 타이밍에 전달합니다.
